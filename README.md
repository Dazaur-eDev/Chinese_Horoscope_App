# Sistema Web para Consultar el Horóscopo Chino

Este sistema web permite a los usuarios registrados consultar el animal representante en el horóscopo chino según su año de nacimiento. La aplicación está construida en Java utilizando Spring Boot y JDBC para la gestión de la base de datos.

## Características

- **Inicio de Sesión y Registro de Usuarios**: Solo los usuarios registrados pueden acceder al buscador del horóscopo chino.
- **Búsqueda del Horóscopo Chino**: Permite consultar el animal del horóscopo chino que corresponde a un año específico.
- **Protección de Acceso**: Los usuarios no registrados no pueden visualizar el buscador.

## Tecnologías Utilizadas

- **Java 17+**: Lenguaje de programación principal.
- **Spring Boot**: Framework utilizado para simplificar el desarrollo y la configuración del sistema.
- **JDBC (Java Database Connectivity)**: Para la conexión y gestión de la base de datos.
- **HTML5/CSS3/Bootstrap5**: Para la interfaz de usuario.

## Instalación

### Prerrequisitos

- Java 11 o superior
- Maven 3.6 o superior
- IDE recomendado: IntelliJ IDEA o Eclipse
